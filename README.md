# Number Guessing Game

A simple console-based number guessing game written in Java.

## How It Works

The program generates a random number between 1 and 100. Your goal is to guess the number before running out of attempts!

After each guess, you'll receive feedback:
- **Too low!** - Your guess is below the target number
- **Too high!** - Your guess is above the target number
- **CONGRATULATIONS!** - You guessed correctly!

## Difficulty Modes

Choose your challenge level:
- **Easy**: 15 attempts
- **Medium**: 10 attempts
- **Hard**: 5 attempts

## How to Run

1. Navigate to the Tycoon directory:
```bash
cd GuessingGame
```

2. Compile the program:
```bash
javac src/App.java
```

3. Run the game:
```bash
java -cp src App
```

## Gameplay

1. Select your difficulty level (1-3)
2. Start guessing numbers between 1-100
3. Use the feedback to narrow down your guesses
4. Try to find the number before running out of attempts!
